Keyboard Shortcuts

Keyboard Shortcuts

If you are a MacOS user, pretend ctrl is actually cmd

  • ctrl+Enter Send Request
  • ctrl+P Switch Requests
  • ctrl+N New Request
  • ctrl+D Duplicate Request
  • ctrl+L Focus URL Bar
  • ctrl+, Show Settings
  • ctrl++ Zoom In
  • ctrl+- Zoom Out
  • ctrl+0 Reset Zoom
