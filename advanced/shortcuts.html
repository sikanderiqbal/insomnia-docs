Keyboard Shortcuts

_Note: If you are on Windows or Linux, meta represents Control and, if you are on Mac, meta represents Command.

  • meta+Enter Send Request
  • meta+P Switch Requests
  • meta+N New Request
  • meta+D Duplicate Request
  • meta+L Focus URL Bar
  • meta+, Show Settings
